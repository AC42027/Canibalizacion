# Sistema de Canibalizaciones - Planta L504

Este es el repositorio oficial para el **Sistema de Control de Canibalizaciones** de Goodyear Planta L504. El sistema permite registrar activos tomados de máquinas donantes para máquinas receptoras, realizar el seguimiento de normalización y monitorear el historial.

## 📂 Organización de Documentación

Toda la documentación técnica, manuales, registros de avances y guías se encuentran organizados en este repositorio:

1. **[README Principal (Raíz)](../README.md)**:
   - Resumen del sistema y arquitectura general.
   - Guía de inicio rápido con el lanzador `INICIAR_SISTEMA.bat`.
   - Integraciones LDAP y SAP.
   - Estructura de base de datos relacional y roles de usuarios.

2. **[Registro de Avances y Guía de Documentación](AVANCES_Y_REGISTROS.md)**:
   - Registro de avances de funcionalidades desarrolladas (árbol de equipos, filtros del dashboard, seguridad, etc.).
   - Credenciales de acceso de prueba por defecto.
   - Parámetros de configuración (tiempos de inactividad de sesión).
   - Guía paso a paso sobre cómo puedes escribir y generar tus propios archivos Markdown `.md`.
